Game Developers in Seattle

Dan Fernandez - MSFT 6/18/2004 4:26:00 PM

Meltdown is a Game Developer Conference in Seattle July 27-28th.  If you're a game developer writing managed or unmanaged applications, this should be a great conference to attend.

Speakers for the VS Track include:

  • Tom Miller on managed Direct X
  • Andy Pennell on debugging
  • Peter Hallam on C#
  • Brandon Bray on C++
  • Kang Su Gatlin on C++ Perf
  • Jason Weber on VS
  • Sean Sandys on VSTS
  • and many more

This should fun :)

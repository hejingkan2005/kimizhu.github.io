Visual Studio Beta 1 SDK & Redist now on MSDN

Dan Fernandez - MSFT 6/30/2004 12:24:00 AM

If you're not interested in using Express or if you want to use the tools available in the .NET Framework SDK (ildasm, wincv, etc), you can download the SDKs (and redists) at the locations provided below:

.NET Framework 2.0 SDK betas (by proc type)

  • .NET Framework 2.0 SDK Beta 1 x86
  • .NET Framework 2.0 SDK Beta 1 x64
  • .NET Framework 2.0 SDK Beta 1 IA64

.NET Framework 2.0 Redist betas (by proc type)

  • .NET Framework Version 2.0 Redistributable Package Beta 1 (x86)
  • .NET Framework Version 2.0 Redistributable Package Beta 1 (IA64)
  • .NET Framework Version 2.0 Redistributable Package Beta 1 (x64)

Java Developer Resource Center is Live!

Dan Fernandez - MSFT 6/7/2004 9:43:00 AM

If you're a Java developer who needs to interoperate with, learn about, or migrate to the .NET Framework, MSDN can be a pretty daunting place for information and a lot of the content seems focused on COM to .NET or VB6 to .NET.  Well Brian Keller and I (among others) just launched the Java Developer Resource Center at: http://msdn.microsoft.com/java/ to be a one-stop shop for Java developer resources on MSDN. 

Below is a quick list of the content available on the site

  • Interoperability - What options are available for .NET and J2EE interoperability
    • .NET and J2EE Platform 4-page interoperability poster and Simon Guest's open letter to the community on interoperability 
    • Three samples chapters from Simon’s .NET and J2EE Interoperability Toolkit book
    • MSDN articles on interoperability
  • Learning - Learn or evaluate C# or J# and the .NET Framework
    • C# Language for Java Developers whitepaper
    • Visual Studio .NET 2003 Interactive Simulation
    • Resources to get you started including PAG patterns, tutorials, quickstarts, and other resources on GDN and MSDN
    • Downloads, Downloads, Downloads 
  • Migration - Migrate legacy Java assets to the .NET Framework
    • Java Language Conversion Assistant resources
    • Visual J# .NET resources
    • Java Virtual Machine migration resources

If you have any suggestions or feedback on the site, let us know either through comments on this blog entry or at Java-NET@Microsoft.com

 

Note: Simon Guest's interoperability poster and open letter mentioned above is in the July issue of Dr. Dobbs magazine

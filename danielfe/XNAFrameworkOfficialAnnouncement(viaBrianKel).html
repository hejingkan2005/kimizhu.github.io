XNA Framework Official Announcement (via BrianKel)

Dan Fernandez - MSFT 3/20/2006 1:09:00 PM

As I previously blogged, it's now officially announced: The XNA Framework is a custom version of the CLR and class libraries built to enable execution of managed code on an Xbox 360 and Windows. At the Game Developer Conference, they are showing games written in managed code running on the Xbox 360. Very cool.

More information at the recently updated XNA home page

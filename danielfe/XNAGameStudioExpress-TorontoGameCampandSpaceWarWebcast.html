Toronto Game Camp - November 11th
Daniel Shapiro from Microsoft Canada is hosting a free 1-day Game Camp for students, teachers, and hobbyists in Toronto that will show off how to create games using XNA Game Studio Express. Sign up while there are still seats available! I'd love it if this came to Seattle :)

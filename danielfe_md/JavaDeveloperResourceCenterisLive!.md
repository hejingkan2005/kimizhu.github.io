# Java Developer Resource Center is Live\!

[Dan Fernandez -
MSFT](https://social.msdn.microsoft.com/profile/Dan%20Fernandez%20-%20MSFT)
6/7/2004 9:43:00 AM

-----

<div id="content">

If you're a Java developer who needs to interoperate with, learn about,
or migrate to the .NET Framework, MSDN can be a pretty daunting place
for information and a lot of the content seems focused on COM to .NET or
VB6 to .NET.  Well [Brian Keller](http://blogs.msdn.com/briankel/) and I
(among others) just launched the Java Developer Resource Center at:
<http://msdn.microsoft.com/java/> to be a one-stop shop for Java
developer resources on MSDN.
